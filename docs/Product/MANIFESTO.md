# Manifesto Evol OS

## Acreditamos nas pessoas.

Todo software de RH fala sobre pessoas.

Mas poucos realmente ajudam pessoas.

O Evol OS existe para mudar isso.

---

## Acreditamos que tecnologia deve simplificar.

Nunca complicar.

Se um usuário precisar de treinamento para executar uma tarefa simples,

nós ainda temos trabalho a fazer.

---

## Acreditamos que gestores podem evoluir.

A maioria das pessoas nunca recebeu treinamento em liderança.

Nosso papel não é julgá-las.

Nosso papel é ajudá-las.

---

## Acreditamos que o RH pode ser estratégico.

RH não existe apenas para controlar processos.

Existe para desenvolver pessoas.

Criar culturas.

Fortalecer empresas.

---

## Acreditamos na IA.

Mas acreditamos ainda mais nas pessoas.

A IA orienta.

Explica.

Resume.

Recomenda.

Ensina.

A decisão continua sendo humana.

---

## Acreditamos que software também educa.

Cada tela deve ensinar algo.

Cada fluxo deve desenvolver o usuário.

Cada interação deve aumentar a maturidade da empresa.

---

## Acreditamos na simplicidade.

O usuário nunca deve perguntar:

"Como faço isso?"

A interface deve responder antes.

---

## Acreditamos em linguagem humana.

Não escrevemos para programadores.

Escrevemos para gestores.

Para colaboradores.

Para pessoas.

---

## Acreditamos que empresas evoluem quando pessoas evoluem.

Esse é o motivo pelo qual o Evol OS existe.

Todo botão.

Toda tela.

Toda funcionalidade.

Toda linha de código.

Deve aproximar a empresa desse objetivo.

---

# Nosso lema

Tecnologia que ensina.

IA que orienta.

RH que evolui.
