# UX Specification

# Jornada

Home de Avaliações

---

# Problema

Ao acessar o módulo de Avaliações, o RH precisa entender rapidamente:

- o que exige atenção;
- como iniciar uma nova avaliação;
- qual é a situação atual.

A interface atual é orientada por tabelas e cadastros.

Ela comunica armazenamento.

Não comunica decisão.

---

# Objetivo

Transformar a Home em um centro de decisão.

---

# Usuários

- RH
- Business Partner
- Gestores

---

# Perguntas que a tela responde

1. O que merece minha atenção hoje?

2. Existe alguma avaliação em andamento?

3. Quantas avaliações estão acontecendo?

4. Como começo uma nova avaliação?

5. Quais modelos já possuo?
